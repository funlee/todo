react-todo-mvc
===============
预览地址：[react-todo](http://show.funlee.cn/react-todo/index.html "react-todo")

安装
----
```bash
git clone https://github.com/funlee/react-todo-mvc.git
cd react-todo-mvc
npm install
npm run start
```
然后在浏览器里输入：http:localhost:8080 即可访问

主要功能
-------
* Task 增加、删除
* 增加 Task 支持回车（Enter）
